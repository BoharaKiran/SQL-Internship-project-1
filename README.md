# Online Retail Sales Database Design

## Project Overview
A fully normalized relational database for an e-commerce platform built using MySQL.
Manages customers, products, orders, and payments efficiently.

## Objective
Design a normalized SQL schema for an online retail platform with analytical reporting.

## Queries Written
- Total Sales Revenue
- Top Selling Products
- Orders Per Customer
- Revenue by Category
- Monthly Sales Trend
- Sales Report VIEW (Bonus)
